# MCP Integration

IncidentFlow MCP exposes a set of tools compatible with the Model Context Protocol.

These tools allow AI agents to:

- fetch incidents
- inspect signals
- retrieve historical patterns
- analyze system health

Example use cases:

- AI-powered incident triage
- automated root cause suggestions
- incident summarization
- proactive alert analysis

## Example interaction

An AI agent can:

1. Receive a new alert
2. Query IncidentFlow MCP
3. Retrieve similar past incidents
4. Suggest likely root cause
5. Recommend mitigation steps

## Quick start example

```bash
curl https://mcp.incidentflow.io/install.sh | sh
```

Then connect your AI agent to:

- retrieve incidents
- analyze alerts
- automate diagnostics
